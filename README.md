# Uni-Vision — Progress Report

A learning portfolio documenting progress through the Uni_Vision 
visual intelligence system curriculum, covering computational thinking, 
Python, web development, APIs, visual workflows, and computer vision.

---

## Week 1 — Computational Thinking
**Focus:** Variables, loops, conditions, events, pipelines  
**Deliverables:** Camera alert flowchart, concept notes  
**Key learning:** A vision system is a pipeline of small decisions, not one big model.  
 [Week 1 folder](./week1/)

## Week 2 — Python Foundations
**Focus:** Functions, dictionaries, lists, type hints, tests  
**Deliverables:** Detection filter module, pytest test suite  
**Key learning:** Tests prove behavior before systems get complicated.  
 [Week 2 folder](./week2/)

## Week 3 — Web Basics and Dashboard Thinking
**Focus:** HTML, CSS, JavaScript, state, events  
**Deliverables:** 3-stage pipeline dashboard with live event log  
**Key learning:** UI state must be correct before polish matters.  
 [Week 3 folder](./week3/)

## Week 4 — APIs and JSON Contracts
**Focus:** FastAPI, Pydantic, HTTP, validation, structured responses  
**Deliverables:** /health and /analyze routes, API contract doc  
**Key learning:** Define the contract before implementation.  
 [Week 4 folder](./week4/)

## Week 5 — Blocks, Graphs, and Execution Order
**Focus:** Visual blocks, ports, DAGs, topological sorting, validation  
**Deliverables:** block_graph.json, graph validation checklist  
**Key learning:** Graph validation must happen before execution.  
 [Week 5 folder](./week5/)

## Week 6 — Images as Arrays
**Focus:** Pixels, channels, shape, resize, grayscale, crop, normalize  
**Deliverables:** Colab notebook with full image pipeline  
**Key learning:** Every model input is just numbers in a useful shape.  
 [Week 6 folder](./week6/)

## Week 7 — Preprocessing and Frame Efficiency
**Focus:** Blur, thresholding, edges, frame sampling, low-light policy  
**Deliverables:** Preprocessing notebook, low-light policy design  
**Key learning:** Preprocessing often matters as much as the model itself.  
 [Week 7 folder](./week7/)
